how to use

Step: 1 Download bee-icon from download link in the top right and extract zip

Step: 2 Link bee-icon.css into your HTML Doucment

  <head>
      <link rel="stylesheet" href="bee-icon.css">
  </head>

OR

Use CDN LINK

   <link rel="stylesheet" href="http://beeknock.com/tools/bee-icon/beeicon-font.css">

That's it, Now you are ready to use beatiful and light weight font icon

Use any html tag like

<i class="bee-icon-search" > </i>
